## Description
Demo project integrating MES data with Machine Learning for predictive maintenance using simulated sensor data in Wokwi.  
## Features
- **Simulate sensor data** (vibration, temperature, power) in Wokwi
- **Collect and standardize data for ML**
- **Train predictive maintenance models** (Random Forest, SVM, Neural Networks)
- **Visualize results with dashboards**
## How to Run 
1. Run sensor simulation in Wokwi
2. Collect data in CSV / database
3. Run Python scripts to preprocess data and train ML models
4. Visualize results in dashboards

## คำอธิบายสั้น
โปรเจกต์ตัวอย่างการเชื่อมข้อมูล MES กับ Machine Learning เพื่อตรวจสอบและทำนายการซ่อมบำรุง โดยใช้ข้อมูลเซนเซอร์จำลองใน Wokwi
## คุณสมบัติ
- **จำลองข้อมูลเซ็นเซอร์** (การสั่นสะเทือน อุณหภูมิ พลังงาน) ใน Wokwi
- **รวบรวมและสร้างมาตรฐานข้อมูลสำหรับ ML**
- **ฝึกแบบจำลองการบำรุงรักษาเชิงคาดการณ์** (Random Forest, SVM, Neural Networks)
- **แสดงภาพผลลัพธ์ด้วยแดชบอร์ด**
## วิธีการเรียกใช้
1. เรียกใช้การจำลองเซ็นเซอร์ใน Wokwi
2. รวบรวมข้อมูลในรูปแบบ CSV/ฐานข้อมูล
3. เรียกใช้สคริปต์ Python เพื่อประมวลผลข้อมูลเบื้องต้นและฝึกแบบจำลอง ML
4. แสดงภาพผลลัพธ์ในแดชบอร์ด
